# CV Project Description

## Version for Completed Project

Bus Out! — Puzzle Game Design Case Study | 2026

- Created a Game Design Document for a casual puzzle game focused on route planning, passenger matching, and limited-move decision making.
- Designed core gameplay loop, rules, win/lose conditions, level progression, and sample puzzle levels.
- Planned balancing factors such as number of buses, passenger colors, obstacles, parking slots, move limits, and completion time.
- Conducted basic paper-based playtesting to evaluate rule clarity, difficulty progression, and player experience.

Skills: Game Design Document, Core Loop Design, Level Design, Puzzle Mechanics, Player Experience, Balancing, Playtesting
